 Common Questions

Q: Who can create change orders?  
A: Any user with appropriate permissions can create change orders for accepted quotes.

Q: Do clients need to log in to approve?  
A: No! Clients receive a secure approval link via email and can approve without logging in.

Q: What happens after a change order is approved?  
A: The status updates to "APPROVED" and you can execute it to generate invoices/credit notes (Phase 5 feature coming soon).

Q: Can I edit a change order after sending it?  
A: Only DRAFT change orders can be edited. Once sent, you'd need to create a new change order.

Q: How do financial calculations work?  
A: The system compares the original quote total with the new total after changes and shows the delta (increase or decrease).

Q: Where do I find change orders?  
A: In the "Change Orders" menu item, or on individual project and quote detail pages.

 Troubleshooting

Issue: Client didn't receive approval email  
Solution: Check spam folder. Verify email address. Resend from change order detail page.

Issue: Approval link says "expired"  
Solution: Change orders have an expiration date. Create a new change order or extend the validity period.

Issue: Can't find the Create Change Order button  
Solution: Only ACCEPTED quotes can have change orders. Check quote status.
